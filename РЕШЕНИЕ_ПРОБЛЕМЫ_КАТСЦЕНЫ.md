# Решение проблемы: Кат-сцена не запускается

## Проблема
Кат-сцена не запускается, независимо от настройки галочки "Skip Intro Cutscene".

## Причина
Используется `MainMenuUI.OnStartGame()`, который правильно настроен и должен загружать кат-сцену. Но возможно:
1. Сцена не загружается из-за неправильного имени
2. Или метод не вызывается

## Что было исправлено в коде:

1. ✅ **Удалены неиспользуемые поля** `initialized` из `LetterSpawner` и `ItemSpawner`
2. ✅ **Удалено неиспользуемое поле** `isTransitioning` из `ImageCutscene`
3. ✅ `MainMenuUI.OnStartGame()` уже настроен правильно и загружает `IntroCutscene`

## Что нужно проверить в Unity Editor:

### 1. Проверьте консоль при нажатии кнопки "Начать"

1. Откройте Unity Editor
2. Откройте консоль: **Window → General → Console**
3. Запустите игру (Play)
4. Нажмите кнопку "Начать"
5. **Посмотрите, какие сообщения появляются в консоли:**

**Если видите:**
- `[MainMenuUI] OnStartGame вызван! Загружаю сцену IntroCutscene...`
- `[MainMenuUI] Сцена IntroCutscene доступна для загрузки`
- `[ImageCutscene] Awake вызван!...`

→ Значит все работает правильно, и кат-сцена должна загружаться.

**Если видите:**
- `[MainMenuUI] Сцена IntroCutscene НЕ доступна!`

→ Значит сцена не в Build Settings (но вы уже добавляли её, так что это странно).

**Если НЕ видите никаких сообщений:**

→ Значит метод `OnStartGame()` не вызывается. Возможные причины:
- Кнопка не связана с `MainMenuUI`
- Скрипт `MainMenuUI` не активен
- Или используется другой скрипт

### 2. Проверьте настройку кнопки "Начать"

1. Откройте сцену `menu.unity`
2. Найдите кнопку "Начать" (Button_start)
3. В инспекторе проверьте компонент `Button`:
   - Убедитесь, что кнопка активна (галочка стоит)
   - Убедитесь, что она интерактивна (Interactable = true)

4. Найдите GameObject `Canvas` (или тот, где находится `MainMenuUI`)
5. Проверьте компонент `MainMenuUI`:
   - Убедитесь, что скрипт активен (галочка стоит)
   - Убедитесь, что поле `Start Button` назначено правильно (ссылается на Button_start)

### 3. Проверьте, не используется ли MenuButtons вместо MainMenuUI

В сцене menu может быть два скрипта:
- `MainMenuUI` - должен использоваться для кнопки "Начать"
- `MenuButtons` - может быть на другом GameObject

Проверьте:
1. Откройте сцену `menu.unity`
2. Найдите GameObject `Main_Panel` (или где находится `MenuButtons`)
3. Если там есть компонент `MenuButtons`, проверьте:
   - Убедитесь, что `Skip Intro Cutscene` = **false** (галочка НЕ стоит)
   - Если галочка стоит - снимите её

### 4. Альтернативное решение: Удалите MenuButtons, если он не нужен

Если `MenuButtons` не используется, можно удалить его из сцены:
1. Найдите GameObject с компонентом `MenuButtons`
2. Удалите компонент `MenuButtons` (правый клик → Remove Component)

## Отладка через код

Если ничего не помогло, добавьте временную отладку прямо в `MainMenuUI.Start()`:

```csharp
private void Start()
{
    Debug.Log("[MainMenuUI] Start вызван!");
    
    // Настройка кнопок
    if (startButton != null) 
    {
        Debug.Log("[MainMenuUI] startButton найден, добавляю слушатель");
        startButton.onClick.AddListener(OnStartGame);
    }
    else
    {
        Debug.LogError("[MainMenuUI] startButton НЕ НАЙДЕН! Проверьте назначение в инспекторе!");
    }
    // ... остальной код
}
```

Это поможет понять, вызывается ли `Start()` и назначена ли кнопка.
