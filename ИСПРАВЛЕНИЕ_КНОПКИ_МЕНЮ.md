# Исправление: Кат-сцена не загружается из меню

## Проблема найдена!

В сцене menu используется скрипт **MenuButtons**, а не MainMenuUI! И в этом скрипте была установлена переменная `skipIntroCutscene = true`, что пропускало кат-сцену.

## Что было исправлено:

1. **Изменил `skipIntroCutscene` на `false`** - теперь кат-сцена не будет пропускаться
2. **Добавил отладочные сообщения** для диагностики
3. **Добавил проверку доступности сцены** через `Application.CanStreamedLevelBeLoaded()`

## Что нужно сделать в Unity Editor:

### Вариант 1: Использовать код (рекомендуется)

Код уже исправлен, просто:
1. Сохраните все файлы (Ctrl+S)
2. Вернитесь в Unity Editor
3. Дождитесь перекомпиляции скриптов
4. Запустите игру и проверьте

### Вариант 2: Изменить вручную в Unity Editor

Если код не применился:

1. Откройте сцену `menu.unity`
2. Найдите GameObject с компонентом `MenuButtons` (скорее всего это Main_Panel или Canvas)
3. В инспекторе найдите компонент `MenuButtons`
4. Убедитесь, что поле **Skip Intro Cutscene** имеет значение **false** (галочка не стоит)
5. Сохраните сцену (Ctrl+S)
6. Запустите игру

## Проверка работы:

1. Запустите игру
2. Нажмите кнопку "Начать"
3. Откройте консоль Unity (Window → General → Console)
4. Вы должны увидеть сообщения:
   - `[MenuButtons] StartGame вызван!`
   - `[MenuButtons] Загружаем кат-сцену: IntroCutscene`
   - `[MenuButtons] Сцена IntroCutscene доступна для загрузки`
   - `[ImageCutscene] Awake вызван! Компонент ImageCutscene инициализируется.`
   - `[ImageCutscene] Start вызван. Проверяю настройки...`

Если видите сообщение `[MenuButtons] Сцена IntroCutscene НЕ доступна!` - проверьте, что сцена действительно в Build Settings.

## Дополнительная информация:

В сцене menu есть ДВА скрипта для работы с кнопками:
- **MenuButtons** - используется для кнопки "Начать" (этот скрипт я исправил)
- **MainMenuUI** - используется для других элементов меню

Если вы хотите использовать MainMenuUI вместо MenuButtons, нужно будет переключить кнопку "Начать" на метод `OnStartGame()` из `MainMenuUI`.
