# БЫСТРАЯ ИНСТРУКЦИЯ: Исправление префаба Letter

## Проблема
Записки не триггерятся при приближении игрока.

## Что нужно сделать в Unity Editor (5 минут)

### 1. Откройте префаб Letter
- Найдите `Assets/Prefabs/Items/Letter.prefab`
- Дважды кликните для редактирования

### 2. Добавьте Rigidbody2D (ОБЯЗАТЕЛЬНО!)
- Выберите корневой GameObject префаба
- Add Component → Physics 2D → Rigidbody 2D
- Настройки:
  - Gravity Scale: **0**
  - Freeze Rotation Z: **✓**

### 3. Исправьте размер коллайдера
- Найдите компонент **Box Collider 2D**
- Убедитесь, что **Is Trigger = ✓**
- Измените **Size** с `0.0001 x 0.0001` на `1 x 1`

### 4. Сохраните префаб
- Нажмите "Apply" в режиме редактирования префаба

## Готово!
Теперь записки должны работать при приближении игрока.
