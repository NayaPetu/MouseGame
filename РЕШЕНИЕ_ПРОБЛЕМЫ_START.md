# Проблема: Start() не вызывается!

## Симптомы:
- ✅ `Awake()` вызывается
- ❌ `Start()` НЕ вызывается
- Сцена сразу переходит к main через секунду

## Возможные причины:

### 1. GameObject становится неактивным между Awake и Start
Unity вызывает `Start()` только для активных GameObject'ов. Если GameObject становится неактивным между Awake и Start, Start не вызовется.

### 2. Проблема с Canvas или родительским объектом
Если родительский объект (например, Canvas) неактивен, Start может не вызваться.

### 3. Сцена перезагружается слишком быстро
Возможно, что-то сразу загружает другую сцену, не давая Start вызваться.

## Что нужно проверить в Unity:

### 1. Проверьте иерархию GameObject'а ImageCutscene:
- Убедитесь, что все родительские объекты активны (Canvas, Canvas Group и т.д.)
- Убедитесь, что сам GameObject ImageCutscene активен

### 2. Проверьте, нет ли скриптов, которые деактивируют GameObject:
- Поищите в сцене IntroCutscene скрипты, которые могут вызывать `SetActive(false)` на ImageCutscene

### 3. Запустите игру и посмотрите новые отладочные сообщения:
После следующего запуска вы должны увидеть:
- `[ImageCutscene] OnEnable вызван!` - это покажет, вызывается ли OnEnable
- Информацию об активности GameObject
