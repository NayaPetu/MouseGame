# КРИТИЧЕСКАЯ ПРОБЛЕМА: Сцена IntroCutscene не добавлена в Build Settings!

## Проблема найдена!

Сцена `IntroCutscene` **НЕ ДОБАВЛЕНА в Build Settings**, поэтому Unity не может её загрузить!

## Решение (ОБЯЗАТЕЛЬНО СДЕЛАТЬ!)

### Шаг 1: Откройте Build Settings

1. В Unity Editor: **File → Build Settings** (или нажмите **Ctrl+Shift+B**)

### Шаг 2: Добавьте сцену IntroCutscene

1. **Откройте сцену IntroCutscene:**
   - В Project окне найдите `Assets/Scenes/IntroCutscene.unity`
   - **Дважды кликните** на неё, чтобы открыть

2. **В окне Build Settings:**
   - Убедитесь, что сцена `IntroCutscene` открыта (вкладка сцены активна)
   - Нажмите кнопку **"Add Open Scenes"** (в правом верхнем углу окна Build Settings)
   - Или перетащите сцену `IntroCutscene.unity` из Project окна в список "Scenes In Build"

3. **Проверьте список "Scenes In Build":**
   - Должны быть следующие сцены (в порядке):
     - menu (с галочкой ✓)
     - IntroCutscene (с галочкой ✓) ← должна появиться
     - main (с галочкой ✓)
     - EndGame (с галочкой ✓)
     - EndCutscene (опционально, если есть)

4. **Убедитесь, что галочка стоит** ✓ напротив `IntroCutscene`

### Шаг 3: Закройте Build Settings

1. Закройте окно Build Settings
2. Сохраните проект: **File → Save Project** (или Ctrl+S)

### Шаг 4: Проверьте работу

1. Запустите игру
2. Нажмите кнопку "Начать"
3. Теперь должна загрузиться кат-сцена `IntroCutscene`!

## Почему это важно?

Unity **не может загрузить сцену**, которая не добавлена в Build Settings. Это защита от случайной загрузки сцен, которые не должны быть в финальной сборке игры.

Когда вы вызываете `SceneManager.LoadScene("IntroCutscene")`, Unity проверяет список сцен в Build Settings. Если сцены там нет - загрузка не происходит (или происходит тихо, без ошибок в некоторых случаях).

## Проверка после исправления

После добавления сцены в Build Settings:

1. Откройте консоль Unity (Window → General → Console)
2. Запустите игру
3. Нажмите "Начать"
4. Вы должны увидеть в консоли:
   - `[MainMenuUI] OnStartGame вызван! Загружаю сцену IntroCutscene...`
   - `[MainMenuUI] Сцена IntroCutscene доступна для загрузки`
   - `[ImageCutscene] Awake вызван! Компонент ImageCutscene инициализируется.`
   - `[ImageCutscene] Start вызван. Проверяю настройки...`

Если видите сообщение `[MainMenuUI] Сцена IntroCutscene НЕ доступна!` - значит сцена все еще не добавлена в Build Settings правильно.
