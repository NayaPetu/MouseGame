# КРИТИЧЕСКАЯ ПРОБЛЕМА: Сцена не загружается правильно!

## Симптомы:
- MenuButtons загружает IntroCutscene
- ImageCutscene.Awake вызывается (значит сцена загрузилась)
- НО GameManager.OnSceneLoaded получает "main" вместо "IntroCutscene"!
- Start() не вызывается

## Возможная причина:

Unity может загружать сцену неправильно, или SceneManager.LoadScene() не работает как ожидается.

## Что я добавил:

1. ✅ Проверку `Application.CanStreamedLevelBeLoaded()` перед загрузкой
2. ✅ Отладку текущей сцены в Start()

## Что нужно проверить:

1. **Сохраните все файлы (Ctrl+S)**
2. **Запустите игру и нажмите "Начать"**
3. **Посмотрите в консоль:**

Должны появиться сообщения:
- `[MenuButtons] Сцена IntroCutscene доступна, загружаю...`
- `[ImageCutscene] Start вызван! Текущая сцена: IntroCutscene` (если Start вызывается)

**Если видите:**
- `[MenuButtons] ОШИБКА: Сцена IntroCutscene НЕ доступна!` → сцена не в Build Settings
- `Текущая сцена: main` в Start → значит сцена не загрузилась правильно

**Сообщите, что показывает консоль!**
