# Исправление: Кат-сцена не показывается при нажатии "Начать"

## Проблема
После нажатия кнопки "Начать" в сцене menu кат-сцена не показывается, сразу загружается main.

## Причина
В скрипте `MainMenuUI.cs` метод `OnStartGame()` напрямую загружал сцену "main", минуя кат-сцену.

## Что было исправлено

✅ Изменен метод `OnStartGame()` в `MainMenuUI.cs`:
- Теперь загружается сцена "IntroCutscene" вместо "main"
- Кат-сцена затем автоматически загрузит сцену "main" после показа всех изображений

## Что нужно проверить в Unity Editor

### 1. Убедитесь, что сцена IntroCutscene существует
- В папке `Assets/Scenes/` должна быть сцена `IntroCutscene.unity`

### 2. Проверьте настройку компонента ImageCutscene в IntroCutscene
- Откройте сцену `IntroCutscene.unity`
- Убедитесь, что:
  - Есть GameObject с компонентом `ImageCutscene`
  - Настроен Image Display (UI Image для отображения)
  - Настроен список Images с изображениями
  - **Next Scene Name:** должен быть `main`

### 3. Настройте кат-сцену (если еще не настроена)
Следуйте инструкции в файле `ИНСТРУКЦИЯ_КАТСЦЕНЫ.md`

## Готово!

Теперь при нажатии кнопки "Начать" в меню:
1. Загрузится сцена `IntroCutscene`
2. Покажутся все изображения кат-сцены
3. После последнего изображения автоматически загрузится сцена `main`

## Примечание

Если вы хотите, чтобы можно было включать/выключать кат-сцену из меню, можно добавить настройку в MainMenuUI (аналогично MenuButtons со skipIntroCutscene), но по умолчанию теперь кат-сцена всегда показывается.
