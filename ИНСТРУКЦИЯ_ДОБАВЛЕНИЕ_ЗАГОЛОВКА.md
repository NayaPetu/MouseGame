# Инструкция: Добавление заголовка в UI письма

## Что было сделано в коде

1. ✅ Добавлено поле `title` в класс `LetterData`
2. ✅ Добавлено поле `letterTitle` типа `TextMeshProUGUI` в класс `LetterUI`
3. ✅ Обновлен метод `ShowLetter`, чтобы устанавливать заголовок

## Что нужно сделать в Unity Editor

### Шаг 1: Добавьте TextMeshProUGUI для заголовка в UI панели письма

1. Откройте сцену `main.unity`
2. Найдите UI панель письма `LetterPanel` (она должна быть в Canvas)
3. Внутри `LetterPanel` создайте новый GameObject для заголовка:
   - Правой кнопкой на `LetterPanel` → UI → Text - TextMeshPro
   - Или: Create → UI → Text - TextMeshPro
   - Назовите его `LetterTitle` (или другое удобное имя)

### Шаг 2: Настройте TextMeshProUGUI для заголовка

1. Выберите созданный `LetterTitle` GameObject
2. В инспекторе настройте компонент TextMeshProUGUI:
   - **Text:** можно оставить пустым (будет устанавливаться кодом) или написать "Заголовок" для тестирования
   - **Font Size:** рекомендуется сделать больше, чем у текста письма (например, 24-32)
   - **Font Style:** можно сделать Bold (жирный)
   - **Alignment:** Center (по центру) или Left (слева) - на ваше усмотрение
   - **Color:** выберите цвет, который отличается от текста письма

### Шаг 3: Расположите заголовок в панели

1. Расположите `LetterTitle` в верхней части панели `LetterPanel`
2. Убедитесь, что он находится выше текста письма (`letterText`)
3. Настройте позицию и размер через RectTransform

### Шаг 4: Назначьте заголовок в компоненте LetterUI

1. Найдите GameObject с компонентом `LetterUI` (обычно это Canvas или его дочерний объект)
2. В инспекторе найдите компонент `LetterUI`
3. В поле **Letter Title** перетащите созданный `LetterTitle` GameObject (компонент TextMeshProUGUI)
4. Сохраните сцену (Ctrl+S)

### Шаг 5: Добавьте заголовки к письмам в LetterManager

1. Найдите GameObject с компонентом `LetterManager` в сцене `main.unity`
2. В инспекторе найдите компонент `LetterManager`
3. Для каждого письма в списках `Main Floor Letters` и `Basement Letters`:
   - Найдите поле **Title** (оно должно появиться после обновления кода)
   - Введите заголовок для каждого письма (например, "Письмо #1", "Важное сообщение" и т.д.)

## Готово!

Теперь при открытии письма будет отображаться заголовок вверху панели, а затем текст письма.

## Пример структуры UI панели письма

```
LetterPanel
├── LetterTitle (TextMeshProUGUI) - ЗАГОЛОВОК
├── LetterText (TextMeshProUGUI) - ТЕКСТ ПИСЬМА
└── CloseButton (Button) - КНОПКА ЗАКРЫТИЯ
```

## Важные замечания

- Если поле `Title` в LetterData пустое, будет использоваться заголовок по умолчанию: "Письмо друга"
- Заголовок должен быть типа TextMeshProUGUI (не обычный Text!)
- Убедитесь, что порядок элементов в UI правильный: заголовок выше текста
